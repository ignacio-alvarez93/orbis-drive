ORBIS DRIVE - MANIFIESTO FUNDACIONAL
Fecha: 26/03/2026
Orbis Drive es una plataforma de inteligencia de mercado automovilístico a escala España-UE.
Surge como evolución de Mercado Ibiza, heredando su arquitectura y principios, pero ampliando
su alcance hacia un sistema multimarca y multipaís.
Arquitectura base:
SCRAPERS → DICT LIMPIO → IIG → DVL → PIPELINE → BASE DE DATOS → ANALÍTICA
Objetivo:
Transformar datos desestructurados en inteligencia de mercado para usuarios (B2C) y empresas
(B2B).
Principios:
- Separación de responsabilidades
- Mejor NULL que dato incorrecto
- Trazabilidad completa
- No inferencia sin base
- Robustez sobre velocidad
Capacidades:
- Análisis de precios y tendencias
- Evaluación de vehículos
- Inteligencia de concesionarios
- Comparativas entre países
Estructura organizativa:
- Dirección General
- Dirección de Scraping
- Dirección de Base de Datos
- Dirección de QA
- Dirección de Datos
- Dirección de Mercado
- Marketing (futuro)
Foco inicial:
Seat Ibiza en España
Liderazgo:
Ignacio Álvarez Cañal
Regla fundamental:
Si una decisión rompe la coherencia del sistema, es incorrecta
